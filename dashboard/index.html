매월 실행 체크리스트

  1. 월말 기준 주도 섹터 상위 2개 확인
  2. 각 섹터 1등 종목 1개씩 확인
  3. 일봉, 4시간봉, 1시간봉 차트로 진입 타이밍 확인
  4. 신규 후보 2개를 동일 비중으로 매수
  5. 6개월 도달 시 50% 매도, 나머지 50%는 주봉 추세로 연장 여부 확인

매수 규칙

  • 전략명: Leader2 One Each
  • 월 신규 매수: 2개
  • 선정 방식: 주도 섹터 상위 2개에서 각각 1등 종목
  • 표시 상태: 월간 매수 후보, 차트 확인 후 진입

보유와 매도

  • 기본 보유: 6개월
  • 6개월 도달: 보유분 50% 매도
  • 연장 조건: 남은 50%는 주봉 10주선 위 + RSI 50 이상이면 최대 12개월까지 연장
  • 연장 종료: 10주선 2주 연속 이탈 또는 최대 보유기간 도달
  • 기본 손절: 없음
  • 실제 악재나 급락장은 별도 수동 검토

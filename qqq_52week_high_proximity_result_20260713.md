# QQQ 52주 최고가 근접도 검증 결과

작성일: 2026-07-13
판정: **탈락, 현재 전략 변경 없음**

## 한 줄 결론

52주 최고가에 가장 가까운 종목을 고르면 하락 폭은 줄었지만 수익도 더 크게 줄었다. 기존의 12-1개월 장기 모멘텀 순위를 대체할 수 없다.

## 무엇만 바꿨나

두 계좌는 같은 날, 같은 QQQ 과거 구성 종목, 같은 가격과 비용으로 거래했다. 9종목, QQQ 25%, 종목 바구니 75%, 개별 종목 15% 상한, 월말 교체, 주간 200일선 이탈 매도 규칙도 같았다.

- 기존 방식: 최근 한 달을 뺀 지난 1년 수익률이 높은 순서
- 새 후보: 현재 가격이 최근 252거래일 최고가에 가까운 순서

그 밖의 조건은 바꾸지 않았다. 2025년 이후 데이터는 후보 선택에 열지 않았다.

## 핵심 결과

| 구간 | QQQ 연복리 / 최대 낙폭 | 기존 장기 모멘텀 | 52주 최고가 근접도 |
|---|---:|---:|---:|
| 개발 2010-08~2021 | 22.56% / -28.56% | **26.12% / -33.45%** | 18.11% / **-27.99%** |
| 검증 2022~2024 | 9.03% / -34.83% | **10.91% / -37.12%** | 5.77% / **-23.99%** |
| 2010-08~2014 | 23.02% / **-16.06%** | **26.68% / -20.47%** | 22.47% / -19.32% |
| 2015~2018 | 11.66% / -22.80% | **14.59% / -26.36%** | 3.18% / **-20.97%** |
| 2019~2021 | 37.84% / **-28.56%** | **42.74% / -33.45%** | 34.53% / -27.99% |
| 2022~2024 | 9.03% / -34.83% | **10.91% / -37.12%** | 5.77% / **-23.99%** |

`연복리`는 매년 같은 비율로 불어났다고 환산한 수익률이다. `최대 낙폭`은 투자 중 고점에서 가장 많이 내려간 폭이며, 0에 가까울수록 방어가 잘됐다는 뜻이다.

## 어떻게 해석하나

새 후보는 기존 방식보다 개발 구간에서 연 8.02%p, 검증 구간에서 연 5.13%p 뒤졌다. QQQ와 비교해도 여섯 평가 구간에서 모두 연복리 수익률이 낮았다. 반면 최대 낙폭은 기존 방식보다 대부분 좋아졌다.

즉, 52주 최고가 근접도는 `강한 주도주를 더 잘 고르는 기준`이라기보다 `가격이 안정적으로 고점 부근에 있는 종목을 고르는 기준`에 가까웠다. 정상적인 조정을 거친 장기 승자를 놓치고, 월마다 고점 근처 종목 사이의 순위가 바뀌면서 거래 횟수도 기존 2,359회에서 2,892회로 늘었다.

## 신뢰성 확인

- 기존 계좌의 개발 26.12%, 검증 10.91%가 앞선 교정 연구와 일치했다.
- 과거 시점별 QQQ 구성 종목 247개와 5거래일 지연 정보를 사용했다.
- 현금 비중은 0%로 유지됐고 편도 비용 0.25%를 반영했다.
- 상장폐지 이벤트 9건이 포함됐다.
- 동일 소스를 두 번 실행한 결과가 소수점 여섯 자리까지 일치했다.

## 결정

52주 최고가 근접도 단독 순위는 탈락시킨다. 초기 후보가 합격할 때만 하기로 사전등록한 거래비용 추가 스트레스와 거래량 결합은 진행하지 않는다. 공개 대시보드, 신호 API, Android 앱의 운용 규칙은 바꾸지 않는다.
